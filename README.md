## 🚀 Objective

To analyze user behavior, content performance, and marketing campaign effectiveness in Southeast Asia (SEA) and identify actionable growth opportunities.

---

## 🧠 Project Overview

This project simulates a real-world product analytics system for a streaming platform.

It covers the **complete data lifecycle**:

* Data generation (multi-table relational dataset)
* Data cleaning & feature engineering (Python)
* Business analysis (SQL)
* Data visualization (Tableau)

---

## 🛠️ Tech Stack

* Python (Pandas)
* SQL (SQLite)
* Tableau Public
* CSV Data Modeling

---

## 📊 Dataset Description

The dataset consists of 5 relational tables:

* Users (demographics & subscription)
* Content (genre & type)
* User Activity (watch behavior)
* Campaign (marketing data)
* Campaign Performance (clicks & conversions)

A final denormalized dataset was created for efficient visualization.

## 💡 Key Insights

* High engagement concentrated in specific SEA regions
* A few genres dominate total watch time
* Premium users contribute disproportionately to engagement
* Certain marketing channels deliver higher conversion rates

## 🚀 Business Recommendations

* Focus content investment on top-performing genres
* Expand aggressively in high-engagement regions
* Optimize low-performing marketing channels
